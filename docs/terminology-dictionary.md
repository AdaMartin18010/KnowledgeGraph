## 📋 翻译规范 / Translation Guidelines

### 1. 基本原则 / Basic Principles

1. **准确性** / Accuracy
   - 翻译必须准确传达原意
   - 避免歧义和误解
   - 保持技术含义的一致性

2. **一致性** / Consistency
   - 同一术语在整个项目中保持一致
   - 避免同义词混用
   - 建立术语对照表

3. **可读性** / Readability
   - 英文表达自然流畅
   - 符合学术写作规范
   - 便于国际用户理解

4. **专业性** / Professionalism
   - 使用标准的学术术语
   - 符合国际学术规范
   - 体现专业水准

### 2. 翻译策略 / Translation Strategy

#### 2.1 直译策略 / Literal Translation

适用于有标准翻译的术语：

- 知识图谱 → Knowledge Graph
- 实体 → Entity
- 关系 → Relation

#### 2.2 意译策略 / Free Translation

适用于需要解释性翻译的术语：

- 批判性分析 → Critical Analysis
- 工程实践 → Engineering Practice
- 形式化方法 → Formal Methods

#### 2.3 音译策略 / Transliteration

适用于专有名词：

- 图论 → Graph Theory
- 语义 → Semantic
- 本体 → Ontology
